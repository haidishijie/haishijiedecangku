# 🀄 胡乐麻将 v0.2.0 - 更新说明

> 更新日期: 2026-06-05
> 提交编号: 053b509

## ✨ 本次更新内容

### 1. 双按钮快速开局
首页新增两个快速开局按钮，功能区分明确：

| 按钮 | 图标 | 颜色 | 特点 |
|------|------|------|------|
| 上方 | 🀄 | 粉色渐变 | 标准模式（最多4人） |
| 下方 | 🃏 | 橘色渐变 | 无上限模式（最多8人） |

### 2. 房间功能永久下线
- 删除 `pages/create-room/`、`pages/join-room/` 共10个文件
- 删除 `utils/room.js`（271行房间管理代码）
- 删除 `cloudfunctions/room/` 云函数
- 清理所有页面中的房间模式逻辑（game/result/index）

### 3. 麻将统计系统（结算页新增）
```
🔥 最大赢家 / 💨 最大输家
📊 胜率 / 📈 平均每轮得分
🏆 庄（赢钱即为庄）
🀄 自摸（仅1赢多输才算）
```

### 4. 分享战绩优化
- 分享截图现在包含完整的麻将统计信息
- 画布高度根据玩家人数自动适配
- 标题改为"胡乐麻战绩"
- 底部水印改为"来自「胡乐麻」打牌记分小程序"
