## 📤 分享功能全面修复

**问题链路：** 分享到朋友圈无入口 → 点了分享 path 指向首页 → 带了 gameId 但本地无数据 → 提示「牌局不存在」

**完整修复：**
1. `game.js` + `result.js` — `onLoad` 启用 `showShareMenu`，开放好友 + 朋友圈两个入口
2. `onShareAppMessage` — `path` 从固定首页改为带 gameId 的结算页
3. `onShareTimeline` — `query` 从空字符串改为 `gameId=${gameId}`
4. 好友收到分享链接 → 直接打开结算页显示完整战绩（同设备）
5. 其他设备打开 → 显示友好提示「该牌局数据仅限原设备查看」
6. `onShare()` actionSheet 新增「分享到朋友圈」选项
